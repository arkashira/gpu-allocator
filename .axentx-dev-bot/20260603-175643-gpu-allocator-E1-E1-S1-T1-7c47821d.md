import random
import time
from typing import Callable, Optional

class DataFeed:
    """
    Simulates (or wraps) a real‑time GPU utilisation source.
    The default `source` is a random uniform generator; in production
    you can inject a callable that reads from Prometheus, nvidia‑smi,
    etc.
    """

    def __init__(self, source: Optional[Callable[[], float]] = None):
        self._source = source or self._random_source
        self._last_value: float = 0.0

    @staticmethod
    def _random_source() -> float:
        """Fallback generator – returns a value between 0 and 100 %."""
        return random.uniform(0.0, 100.0)

    def get_current_usage(self) -> float:
        """
        Returns the most recent utilisation percentage.
        The call is deliberately cheap – it just forwards to the source.
        """
        self._last_value = float(self._source())
        return self._last_value
```

*Why this is better* – the original draft missed the `numpy` import and used `np` without defining it. Here we avoid NumPy altogether for the feed (a simple `random` call is enough) and expose a clean injection point for real metrics.
